# Calculator

A simple web-based calculator built with HTML, CSS and JavaScript.

Features:
- Basic arithmetic operations: addition (+), subtraction (-), multiplication (*), and division (/)
- Decimal support
- Clear and backspace
- Keyboard support for digits and operators

How to run:
1. Clone the repository or download the files.
2. Open index.html in a web browser. No build step or server required.

Usage:
- Click the buttons or use your keyboard to type an expression, then press '=' or Enter to evaluate.
